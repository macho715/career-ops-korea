# JobKorea 안전 사용 가이드라인

> **2026-06-18** | 개인 구직자용 실전 가이드

---

## 핵심 원칙

JobKorea는 경쟁사 크롤링(사람인 판례)에는 적대적이지만, **개인 구직자의 정상적인 이용은 문제없습니다.** 아래 가이드라인은 계정 보호를 위한 실용적 조언입니다.

---

## JobKorea 정책 요약

### robots.txt

```
User-agent: *
Disallow: /Search/       ← 검색 경로 (대량 크롤러 대상)
```

**개인 구직자 관점**: robots.txt는 대량 스크래퍼를 겨냥한 것입니다. 하루 몇 건의 검색은 문제되지 않습니다.

### 이용약관 핵심

> 서비스 정보를 대량 복제·재배포하는 행위 금지  
> 서비스 안정적 운영을 저해하는 행위 금지

**개인 구직자 관점**: 개인이 자신의 구직을 위해 몇 건 검색하고 지원하는 것은 정상적인 서비스 이용입니다.

## 리스크 매트릭스

| 행위 | 위험도 | 이유 |
|------|--------|------|
| **스캔 (하루 1~2회)** | 🟢 LOW | 정상적 이용 패턴 |
| **자동지원 (직접 검토 후)** | 🟢 LOW | 사용자가 최종 결정 |
| **대량 지원 (>10/일)** | 🟡 주의 | 비정상 패턴으로 탐지 가능 |
| **상업적 이용** | 🔴 금지 | 타 서비스 구축 목적 금지 |

## 안전 사용 규칙 (7원칙)

### 1. 시간 간격 — 인간처럼 행동하라

| 파라미터 | 최소 | 권장 |
|----------|------|------|
| 페이지 간 간격 | 3초 | **5~10초** (지터) |
| 검색 간 간격 | 30초 | **60초+** |
| 지원 간 간격 | 5분 | **30분+** |
| 세션당 최대 | 5건 | **3건** |
| 일일 최대 | 10건 | **5건** |

### 2. 헤드리스 금지 — 항상 headed 모드

```bash
# 항상 headed 모드로 실행
node jobkorea-apply.mjs --report 42 --headless=false
```

- Headless Chrome은 탐지됨
- 실제 사용자 브라우저처럼 행동해야 함
- navigator.webdriver = false 설정

### 3. 실제 사용자 행동 시뮬레이션

```
1. 메인 페이지 접속 → 3~5초 대기
2. 검색 페이지 천천히 스크롤 → 2~3초 대기
3. 공고 클릭 → 5~10초 읽는 시늉
4. 지원하기 클릭 → 폼 천천히 작성
5. 제출 전 30초 이상 검토
```

### 4. 로그인 세션 유지

```bash
# 한 번 로그인 후 세션 재사용
# 매번 로그인/로그아웃 반복 금지
# 하루 1~2회 로그인으로 제한
```

### 5. VPN / 프록시 사용 금지

- JobKorea는 VPN 탐지 시스템 운영
- VPN 감지 시 즉시 차단 가능성 있음
- 실제 IP에서만 접속

### 6. 개인 구직 목적으로만 사용

- 절대 상업적 목적 금지
- 절대 타인을 위한 대리 지원 금지
- 절대 데이터 수집/재배포 금지

### 7. 항상 사용자가 최종 결정

```bash
# PREFLIGHT 검토 필수
# 자동 제출 절대 금지
# 모든 지원서 사용자가 직접 확인 후 제출
```

## 권장 워크플로우

```
1일 1회, 오전/오후 1세션
─────────────────────────
세션 시작 (10:00)
├── 1차 검색 (1키워드, 1페이지)  — 15초
├── 5분 대기
├── 2차 검색 (다른 키워드)       — 15초
├── 30분 대기
├── 공고 검토 (pipeline 평가)
├── 지원 1건 (직접 검토 후)      — 5분
├── 30분 대기
└── 지원 2건 (직접 검토 후)      — 5분

일일 최대: 검색 3회, 지원 5건
```

## 절대 하면 안 되는 것

| 행위 | 이유 |
|------|------|
| ❌ `scan.mjs` 연속 실행 | robots.txt 위반 + 탐지 |
| ❌ `jobkorea-apply.mjs` 루프 | 계정 영구정지 |
| ❌ `--headless=true` | 즉시 차단 가능 |
| ❌ 자동 제출 | ICNA 위반 |
| ❌ 새벽 시간 대량 실행 | 이상 패턴 탐지 |
| ❌ 여러 계정 사용 | 부정행위 간주 |
| ❌ 프록시/VPN 사용 | 탐지 시스템에 포착 |

## Saramin과 비교

| 항목 | JobKorea | Saramin |
|------|----------|---------|
| 스캔 방식 | Playwright (브라우저) | 공식 REST API |
| 자동지원 | Playwright headed | Playwright headed |
| API | 없음 | 공식 API (무료) |
| 일일 제한 | 자율 규제 권장 | 500회 (명시적) |
| 추천 용도 | 소량 검색 + 지원 | 대량 검색 + 분석 |

**결론: 대량 검색은 Saramin, 정밀 지원은 둘 다 사용.**
