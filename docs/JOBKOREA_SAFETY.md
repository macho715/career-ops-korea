# JobKorea 안전 사용 가이드라인

> **2026-06-18 조사 기준** | 외부 검색 + 판례 분석

---

## 위험 등급: 🔴 HIGH

JobKorea는 한국에서 자동화/스크래핑에 가장 적대적인 채용 플랫폼입니다.

---

## 법적 프레임워크

### 적용 법률

| 법률 | 적용 대상 | 위반 시 |
|------|----------|---------|
| **정보통신망법 (ICNA)** Art.48 | 자동화 접근, 로그인 우회 | **형사처벌 가능** (2024년 개정) |
| **저작권법** (DB권) | 채용 DB 크롤링 | 손해배상 |
| **부정경쟁방지법 (UCPA)** | 경쟁사 데이터 수집 | 손해배상 + 금지명령 |
| **개인정보보호법 (PIPA)** | 개인정보 수집 | 과징금 |

### 판례

| 사건 | 결과 | 금액 |
|------|------|------|
| **JobKorea v. Saramin** (2017) | JobKorea 승소 | **100억원대** |
| **Yanolja 판례** | 형사 무죄 ≠ 민사 안전 | **10억원 배상** |

## JobKorea 정책 분석

### robots.txt (2026-04-01)

```
User-agent: *
Disallow: /Search/       ← 검색 경로 전면 금지
Disallow: /Recruit/      ← 채용공고 경로 금지
Allow: /                 ← 메인 페이지만 허용
```

**AI 크롤러**: GPTBot, ClaudeBot — 제한적 허용 (비검색 경로만)
**차단 대상**: CCBot, Diffbot, DeepSeekCrawler, Bytespider — 전면 차단

### 이용약관 (2026-01-29)

> Art.18.4: 서비스 정보를 복사·복제·타인에게 제공하는 행위 금지
> Art.18.5.7: 서비스 안정적 운영을 저해하는 행위 금지
> 위반 시: 사전통지 없이 계정 정지 + 자료 삭제 + 손해배상 청구

## 안전 사용 규칙 (7원칙)

### 1. 시간 간격 — 인간처럼 행동하라

| 파라미터 | 최소 | 권장 |
|----------|------|------|
| 페이지 간 간격 | 3초 | **5~10초** (지터) |
| 검색 간 간격 | 30초 | **60초+** |
| 지원 간 간격 | 5분 | **30분+** |
| 세션당 최대 | 5건 | **3건** |
| 일일 최대 | 10건 | **5건** |

### 2. 헤드리스 금지 — 항상 headed 모드

```bash
# 항상 headed 모드로 실행
node jobkorea-apply.mjs --report 42 --headless=false
```

- Headless Chrome은 탐지됨
- 실제 사용자 브라우저처럼 행동해야 함
- navigator.webdriver = false 설정

### 3. 실제 사용자 행동 시뮬레이션

```
1. 메인 페이지 접속 → 3~5초 대기
2. 검색 페이지 천천히 스크롤 → 2~3초 대기
3. 공고 클릭 → 5~10초 읽는 시늉
4. 지원하기 클릭 → 폼 천천히 작성
5. 제출 전 30초 이상 검토
```

### 4. 로그인 세션 유지

```bash
# 한 번 로그인 후 세션 재사용
# 매번 로그인/로그아웃 반복 금지
# 하루 1~2회 로그인으로 제한
```

### 5. VPN / 프록시 사용 금지

- JobKorea는 VPN 탐지 시스템 운영
- VPN 감지 시 즉시 차단 가능성 있음
- 실제 IP에서만 접속

### 6. 개인 구직 목적으로만 사용

- 절대 상업적 목적 금지
- 절대 타인을 위한 대리 지원 금지
- 절대 데이터 수집/재배포 금지

### 7. 항상 사용자가 최종 결정

```bash
# PREFLIGHT 검토 필수
# 자동 제출 절대 금지
# 모든 지원서 사용자가 직접 확인 후 제출
```
